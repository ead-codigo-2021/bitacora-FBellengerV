Introducción

Se explica el lenguaje HTML (Hypertext Markup Language) el cual se usa para la creación de páginas web definiendo una estructura básica, por medio de distintas etiquetas. Actualmente se encuentra en la quinta versión (HTML5).

Back-end y front-end

El back-end es una parte fundamental de cualquier sitio web o aplicación web; en cambio, el front-end es toda la parte visible de las aplicaciones y sitios web.
