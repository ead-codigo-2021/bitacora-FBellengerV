CSS

Hojas de estilo en cascadas (Cascade Style Sheets). Estos archivos permiten mejorar la apariencia y la accesibilidad de las páginas web. Los archivos CSS se asocian en el archivo HTML por medio de la etiqueta link.

- El conjunto es una regla y la propiedad más el valor es una declaración.

- Puede haber múltiples selectores y múltiples declaraciones
